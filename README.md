# Refera
Interpreted scripting language.

## Building
Required components:
- GNU Make
- GCC/Clang
- Yacc
- Lex

Build steps:
1. Run `make`
2. Interpreter executable will be placed at `bin/refera`
3. Interpreter library will be placed at `bin/librefera.so`
